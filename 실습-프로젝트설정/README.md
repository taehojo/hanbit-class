# 실습-프로젝트설정

Part 4. CLAUDE.md와 커스텀 명령어 실습 결과.

상식 퀴즈게임을 만들며 프로젝트 설정 파일과 커스텀 명령어를 활용한 결과물.

## 파일
- `index.html`, `style.css`, `script.js`, `questions.js` - 퀴즈 게임 웹
- `CLAUDE.md` - 프로젝트 설정 파일 (퀴즈 문제 교차 검증 가이드라인)
- `.claude/commands/` - 커스텀 명령어 (`/quiz-check`, `/quiz-range`, `/quiz-validate`)
- `sample-quiz.zip` - 샘플 예제 파일
